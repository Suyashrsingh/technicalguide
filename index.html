PowerPoint

PowerPoint is a presentation program that allows users to create slide shows with multimedia content. It is part of the Microsoft Office suite.

मराठीत: पॉवरपॉईंट हे एक सादरीकरण प्रोग्राम आहे जे वापरकर्त्यांना मल्टीमीडिया सामग्रीसह स्लाइड शो तयार करण्यास अनुमती देते. हे मायक्रोसॉफ्ट ऑफिस सूटचा भाग आहे.

Excel

Excel is a spreadsheet program that lets users organize, format, and calculate data. It is used for business analysis and simple data tracking.

मराठीत: एक्सेल हे स्प्रेडशीट प्रोग्राम आहे जे वापरकर्त्यांना डेटा आयोजित, स्वरूपित आणि गणना करण्यास अनुमती देते. हे व्यवसाय विश्लेषण आणि साध्या डेटा ट्रॅकिंगसाठी वापरले जाते.

Word

Microsoft Word is a word processor that allows users to create professional documents, insert images, and format text.

मराठीत: मायक्रोसॉफ्ट वर्ड हे एक वर्ड प्रोसेसर आहे जे वापरकर्त्यांना व्यावसायिक दस्तऐवज तयार करण्याची, प्रतिमा समाविष्ट करण्याची आणि मजकूर स्वरूपित करण्याची परवानगी देते.

Learn MS Word

Inserting Images

To insert an image into a Word document, select 'Insert', then 'Pictures', and choose the image.

मराठीत: वर्ड दस्तऐवजामध्ये प्रतिमा समाविष्ट करण्यासाठी 'इन्सर्ट' आणि नंतर 'पिक्चर्स' निवडा व प्रतिमा निवडा.

Image Resizing

To resize an image, select the picture and drag a corner handle.

मराठीत: प्रतिमेचा आकार बदलण्यासाठी, प्रतिमा निवडा आणि कोपऱ्यातील हँडल खेचा.

Compressing Images

Select the image, go to the 'Format' tab, and click 'Compress Pictures'.

मराठीत: प्रतिमा निवडा, 'फॉरमॅट' टॅबवर जा आणि 'कॉम्प्रेस पिक्चर्स' वर क्लिक करा.

Learn MS Excel

Data Management

Excel uses a grid of cells to organize data. Each cell can contain text, numbers, or formulas.

मराठीत: एक्सेल डेटा आयोजित करण्यासाठी सेलच्या ग्रीडचा वापर करते. प्रत्येक सेलमध्ये मजकूर, संख्या किंवा सूत्रे असू शकतात.

Pivot Tables

Pivot tables help summarize and analyze large datasets effectively.

मराठीत: पिव्होट टेबल्स मोठ्या डेटासेटचे सारांश आणि विश्लेषण करण्यात मदत करतात.

Exporting to SharePoint

Excel allows exporting tables to SharePoint for collaboration.

मराठीत: एक्सेल टेबल्स शेअरपॉइंटवर निर्यात करण्याची परवानगी देते ज्यामुळे सहयोग शक्य होते.

Learn MS PowerPoint

Creating Presentations

PowerPoint helps create slides with text, images, music, and more for engaging presentations.

मराठीत: पॉवरपॉइंट आकर्षक सादरीकरणांसाठी मजकूर, प्रतिमा, संगीत, आणि अधिकसह स्लाइड्स तयार करण्यात मदत करते.

Transitions and Animations

PowerPoint provides various transitions and animations to make slides dynamic.

मराठीत: पॉवरपॉइंट विविध संक्रमण आणि अॅनिमेशन्स प्रदान करते ज्यामुळे स्लाइड्स गतिशील बनतात.

Slide Master

Slide Master helps maintain a consistent look throughout your presentation by editing slide layouts.

मराठीत: स्लाइड मास्टर तुमच्या संपूर्ण सादरीकरणात सुसंगत रूप राखण्यास मदत करते जे स्लाइड लेआउट संपादन करून होते.
